ASISTENTE DE INVERSIÓN
======================

REQUISITOS
----------
- Python 3.10 o 3.11 recomendado (no uses 3.14, algunas librerías no son compatibles aún)
- Descargar e instalar Python desde: https://www.python.org/downloads/

INSTALACIÓN
-----------
1. Copia los archivos app.py, requirements.txt y cartera.json en una carpeta de tu ordenador.
2. Abre la terminal y navega hasta esa carpeta:
   cd ruta/a/la/carpeta

3. Instala las dependencias con un solo comando:
   pip install -r requirements.txt

4. Ejecuta la app:
   python -m streamlit run app.py

NOTAS
-----
- cartera.json contiene tus posiciones guardadas. Si no lo copias, empezarás con la cartera vacía.
- La app se abrirá automáticamente en tu navegador en http://localhost:8501
